📱 Chapri

A private, modern, and secure messenger built with Flutter.

✨ Overview

Chapri adalah aplikasi chat pribadi yang dibuat menggunakan Flutter dan Firebase. Aplikasi ini dirancang sebagai messenger sederhana namun modern dengan fokus pada kecepatan, privasi, dan kemudahan penggunaan.
